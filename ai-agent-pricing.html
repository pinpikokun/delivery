Claude Code のモデル

汎用言語モデル（Haiku 4.5 / Sonnet 4.5 / Opus 4.5）を使用。自然な日本語での説明が得意で、コードの解説がわかりやすい。

⚙

Codex CLI のモデル

コーディング特化モデル（Codex-Mini / Codex / Codex-Max）を使用。長時間タスクに強いが、日本語の説明がやや簡素な傾向。

利用回数の制限が結構厳しい

どちらのサービスも、プランによって5時間あたりの利用上限が設けられています。ヘビーユースには上位プランの検討を。
