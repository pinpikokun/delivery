⚙

Claude Code のモデル

汎用言語モデル（Haiku 4.5 / Sonnet 4.5 / Opus 4.6）を使用。自然な日本語での説明が得意で、コードの解説がわかりやすい。

⚙

Codex CLI のモデル

コーディング特化モデル（Codex-Mini / Codex / Codex-Max）を使用。長時間タスクに強いが、日本語の説明がやや簡素な傾向。

⚙

GitHub Copilot のモデル

マルチモデル対応（GPT-4o / Claude Sonnet / Gemini 等）。IDE統合が強力で、コード補完からエージェントまで幅広くカバー。

「エージェント対応」と言っても粒度が違います

  • Claude Code / Codex CLI =「プロジェクト全体を理解して自律的に大きなタスクをこなす」フルエージェント
  • Copilot のエージェントモード = IDE内で段階的にコード修正を提案・適用する、比較的軽量なエージェント

$10で無制限に見えるのは、1回の消費が小さいから成り立つビジネスモデルです。
Claude CodeやCodexで同じことをやると、すぐに5時間の上限に達します。
